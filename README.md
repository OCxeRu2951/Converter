# メディア変換・LUFS測定ツール

Node.js と ffmpeg を活用した、インタラクティブなメディア変換・ラウドネス調整ツールです。

* `convert.js`：動画・音声ファイルの形式変換、および LUFS（ラウドネス）の正規化
* `checker.js`：音声ファイルのラウドネス（LUFS）測定

動作には **Node.js** と **ffmpeg** が必要です。

---

## 依存ライブラリ

本プロジェクトは `fluent-ffmpeg` を使用しています。

次のコマンドで一括インストールしてください。

```bash
npm install
```

※ ffmpeg 本体がシステムにインストールされ、パスが通っている必要があります。

---

## 使い方

### 1. convert.js（形式変換・ラウドネス正規化）

スクリプトを実行すると、同じフォルダ内にある対象ファイルを自動でスキャンし、リストを表示します。

1. **ファイル選択**: 表示されたリストから番号を入力してファイルを選択します。
2. **出力設定**: 映像・音声のどちらにするか、拡張子、コーデックなどを対話形式で設定します。
3. **変換**: 元のファイル名を引き継いだ名称（例: `元ファイル名_processed.mp4`）で出力されます。

**実行コマンド:**

```bash
node convert.js
```

**対応フォーマット（自動スキャン対象）:**

* 動画: `.mp4`, `.mov`, `.mkv`, `.webm`
* 音声: `.mp3`, `.wav`, `.flac`, `.aac`

---

### 2. checker.js（LUFS測定）

指定したファイルのラウドネス値を素早くチェックします。

**実行コマンド:**

```bash
node checker.js
```

---

## 主な機能

* **インタラクティブCLI**: コードを書き換えることなく、実行時の対話で設定を完結。
* **自動ファイル検知**: フォルダ内のメディアファイルを自動でリストアップ。
* **Loudnorm正規化**: ターゲットLUFS（デフォルト -14）に合わせた高品質な音量調整。
* **柔軟な出力**: 元のファイル名を保持したまま、最適な形式へ変換。

---

## 動作環境

* Node.js v16 以上推奨
* ffmpeg がインストールされていること
* macOS / Windows / Linux 対応

---

## 注意事項

* LUFS 測定・変換には ffmpeg の `loudnorm` フィルタを利用しています。
* 変換後のファイルは、上書き防止のため末尾に `_processed` が付与されます。
